## Antes de run

Instalar y configurar IONIC en su maquina, es necesario tener node.js

```bash
  npm i -g @ionic/cli
```

## Pasos para correr

Para instalar las dependencias necesarias
```bash
  npm i
  ionic server
```

Si todo sale bien se ejecutará en el puerto 8100.
